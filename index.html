By the way, take a look at these apps I built…

Airline Financial Dashboard

A dashboard that explores several financial performance metrics of major US airlines. Allowing custom filtering, comparisons over time, and reporting period summaries, this app enables quick and intuitive reviews of airline financial metrics. Insights about results and natural language summary of annual performance are generated by LLM integration via OpenAI API. Built with Python (including pandas and Plotly), API integrations, and Streamlit. Deployed with Docker and Google Cloud Run.

Traffic Impact Predictor

An app that delivers a machine learning model developed to predict the impact of an accident on traffic. The app requires only a single click on a map and then fetches all needed input variables to deliver the prediction from the pretrained model. App is the result of complete model pipeline including data cleaning, model training and finetuning, and model deployment. Built with Python (including sklearn, XGBoost, and other statistical libraries), API integrations, and Streamlit.
